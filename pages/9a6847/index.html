Wargaming API 获取

如果你想要使用本机器人大概率是因为 WOWS 查询模块

如果想要申请并使用 WG 的 API 必须先获取一个 KEY

接下来假设你有一个 Wargaming 的账号

这个账号必须绑定手机才能使用

应该不会有 wows 人没有 Wargaming 账号吧

登陆 Wargaming 的开发者 网站 (opens new window)

选择 MY APPLICATIONS 随后选择 + ADD NEW APPLICATION

输入一个恰当的名字 选择 Mobile 选项后即可申请
