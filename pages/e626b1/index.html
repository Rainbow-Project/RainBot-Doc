# 对于 Windows 与 MacOS

版本注意

建议使用 Python3.10.0 以上的版本

在 Python 官网 (opens new window) 上下载对应版本

小提示

如果你不知道该下那个版本 可以下载 Python3.10.3 在笔者的电脑上运行良好

# 对于 Linux 与 MacOS

      sudo apt-get update
  sudo apt-get install python3.10.3

sudo pacman -S python3.10.3

    brew install python3
